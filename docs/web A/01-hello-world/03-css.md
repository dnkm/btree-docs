# CSS - The Style

Now that we have our skeleton, let's make it look good!
**CSS** (Cascading Style Sheets) is the code that styles the web page.

## Adding Color

We can change the color of our text using CSS.
We will use the `style` attribute for now.

### Example

Try changing the colors to `green`, `purple`, or hex codes like `#ff0000`.

```html iframe height=10lh showEditor
<h1 style="color: blue;">Hello World</h1>
<p style="color: red;">This text is red.</p>
```

## Changing Fonts

We can also change the size of the text using `font-size`.

```html iframe height=5lh showEditor
<p style="font-size: 20px;">This text is bigger!</p>
```
