# Project 1 Exercises

Here are some challenges to practice what you've learned.

## Exercise 1: Rainbow Text

**Objective**: Create a word where each letter is a different color.
**Hint**: Wrap each letter in a `<span>` tag and style it.

**Expected Result**:

```html iframe height=5lh
<span style="color: red">H</span>
<span style="color: orange">e</span>
<span style="color: yellow">l</span>
<span style="color: green">l</span>
<span style="color: blue">o</span>
```

## Exercise 2: User Input

**Objective**: Use JavaScript to ask the user for their name and then display "Hello [Name]!".
**Hint**: Use `prompt()` to get the name and `document.write()` or `innerHTML` to show it.

**Expected Result**:
_(Note: The result below shows what happens after entering a name)_

```html iframe height=5lh
<h1>Hello Student!</h1>
```
