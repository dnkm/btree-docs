# Exercise: Writing Content

Now it's your turn to write some code!

## Goal

Create a simple page with a heading and two paragraphs.

## Instructions

1. Create an `<h1>` tag with the text "About Me".
2. Create a `<p>` tag describing who you are.
3. Create another `<p>` tag describing what you like to do.

## Desired Outcome

```html iframe height=10lh
<h1>About Me</h1>
<p>I am a coding student.</p>
<p>I love building websites.</p>
```
