# Quiz: HTML Basics

Test your knowledge on what we just learned!

1. What does HTML stand for?

   - [ ] HighTech Markdown Language
   - [x] HyperText Markup Language
   - [ ] HyperText Make Language

2. Which tag is used for the main heading?

   - [ ] `<head>`
   - [ ] `<heading>`
   - [x] `<h1>`

3. What symbol is used to close a tag?
   - [x] `/` (e.g. `</h1>`)
   - [ ] `\` (e.g. `<\h1>`)
   - [ ] `.` (e.g. `<.h1>`)
